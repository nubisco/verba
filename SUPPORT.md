# Support

## Community support (free)

- **GitHub Issues**: bug reports and feature requests: [github.com/nubisco/verba/issues](https://github.com/nubisco/verba/issues)
- **GitHub Discussions**: questions, ideas, and general help: [github.com/nubisco/verba/discussions](https://github.com/nubisco/verba/discussions)

Please search existing issues and discussions before opening a new one.

## Self-hosting help

The [Self-Hosting guide](./docs/self-hosting.md) covers Docker Compose setup, environment variables, reverse proxy configuration, and upgrades.

## Commercial support

Enterprise Edition license holders and managed SaaS customers can contact Nubisco Lda directly:

- **Email:** [support@nubisco.io](mailto:support@nubisco.io)
- **Licensing:** [licensing@nubisco.io](mailto:licensing@nubisco.io)

## Security issues

Do not open public issues for security vulnerabilities. See [SECURITY.md](./SECURITY.md).
